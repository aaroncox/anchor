## Running development mode

1. `git clone git@github.com:aaroncox/anchor.git anchor`
2. `cd anchor`
3. `yarn install`
4. `yarn dev`

## Building OS native binary

### all binaries

1. `git clone git@github.com:aaroncox/anchor.git anchor`
2. `cd anchor`
3. `yarn install`
4. `yarn package-all`

### macOS binaries

1. `git clone git@github.com:aaroncox/anchor.git anchor`
2. `cd anchor`
3. `yarn install`
4. `yarn package`

### windows binaries

1. `git clone git@github.com:aaroncox/anchor.git anchor`

4. `yarn package-win`

### linux binaries
